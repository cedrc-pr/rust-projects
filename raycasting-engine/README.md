# Helper

To run it:

```shell
cargo run --example runner
```

To navigate:

- `z`: go forward
- `q`: go left
- `s`: go backward
- `d`: go right
- `a`: turn left
- `e`: turn right
- `ArrowDown`: pause the game

# Map

the map is based on a file with `#` as walls and spaces to be able to walk in the maze.
